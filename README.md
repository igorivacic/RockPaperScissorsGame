# Installation
Run the following in your project dir
* bundle install

# Running tests
Run the following in your project dir
* bundle exec rspec

# Usage
Run the following in your project dir
* rails s
* bin/webpack-dev-server

Open your browser, and navigate to http://localhost:3000
